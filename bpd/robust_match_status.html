How this dashboard works

Every cell is generated from automated tests. No manual annotation. Two referees feed the dashboard:
• bpd/bench/stanford_referee.py — authoritative L1 judge using the actual Stanford KernelBench problem files as reference (prob.Model().forward()). Runs the Stanford gate (canonical shapes 64×1024, 256×1024) plus a robust gate (size sweep + edge-case vector: ±0, ±1, ±88, denormals, ±inf, NaN).
• bpd/tests/verify_robust_ops_auto.py — Iyun's harness emitting structured Prolog facts with coordinates_pinned/1 evidence (the parameters that closed each cell: form, signed_zero_branch, accumulator dtype).

Outputs of both referees flow through bpd/tools/wire_referees_to_dashboard.py into bpd/lib/robust_op_match.o.pl, the dashboard's source-of-record.

Color meanings (closed-form, no human gating):
🥇 Gold — 0 ULP across the size sweep AND all edge-case checks pass. Bit-exact AND robust.
🥈 Silver — 0 ULP canonical, but at least one robustness check fails (e.g., -0.0 doesn't round-trip correctly).
🥉 Bronze — within 4 ULP across the sweep, or some shapes diverge. Empirically close but not bit-exact.
⚫ Dim grey — robustness harness has not yet measured this cell.

The parameter-extraction framing (Heath, 2026-05-31): every structural fact lifted from asm becomes a parameter of the implementation. Unroll factor, transcendental identity, signed-zero branch, accumulator dtype — each surfaces a coordinate the generator must pin to close a cell. The drill-down for each cell shows the coordinates_pinned/1 evidence — exactly which parameters closed that cell.

Source of truth: bpd/lib/robust_op_match.o.pl (generated facts from referees) + bpd/robust_match_status.pl (SVG generator). Regenerate via swipl -g main bpd/robust_match_status.pl. Pure deduction from facts.

Canonical addressing: Table(10100).Robust_0ULP_Match_Status. Each cell carries a watermarkup coordinate: data-dim="cell(table(10100),column(<ref>),row(<pattern>))".

Relation to Table(10001): 10001 tracks raw ULP (0 ULP on canonical input). 10100 raises the bar: 0 ULP across a size sweep + edge-case stability. A cell gold in 10001 may be silver or bronze in 10100 until robustness is verified.
